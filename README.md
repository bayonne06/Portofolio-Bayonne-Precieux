## 📌 Sections

- **Header** — Nom, titre, photo de profil et navigation
- **Projets** — Projet actuel (ce portfolio) + projets futurs (*takeROOM*, *MainEvent*)
- **Compétences** — Langages, outils et soft skills
- **Informations** — Localisation, langues, formation et centres d'intérêt
- **Contact** — Formulaire avec champs nom, email, téléphone, sujet et message

## 💡 Projets futurs présentés

Application web de réservation de salles de réunion, conférences et événements.

### 🎉 MainEvent
Réseau social dédié aux événements et à la vente de tickets en ligne.
**Stack prévue :** HTML · CSS · JavaScript · Node.js · Next.js · Prisma · PostgreSQL · Tailwind CSS
